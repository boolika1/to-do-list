About This Project

TaskFlow is a simple and efficient To-Do List web application designed to help students manage their daily tasks and stay organized.

This project was developed by students from the Faculty of Computers and Information as part of a front-end web development course.

Our Goal

  • Help users organize their daily tasks
  • Improve productivity and time management
  • Provide a clean and easy-to-use interface

Features

  • Add and delete tasks
  • Responsive design

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
